# Overview

Built the first automated quotation system for 3D printing services in Poland. The system analyzes 3D models, calculates printing time, material costs, and generates instant quotes.

## Technical Stack

- **Backend**: PHP with MySQL
- **Frontend**: JavaScript with custom UI
- **3D Processing**: Custom algorithms for STL file analysis
- **Automation**: Automatic price calculation based on model parameters

## Features

- Instant quote generation
- Material cost optimization
- Printing time estimation
- Multi-material support

## Impact

Reduced quote turnaround time from hours to seconds, significantly improving customer experience and business efficiency.
