# Opis

Zbudowałem pierwszą automatyczną wyceniarkę dla usług druku 3D w Polsce. System analizuje modele 3D, oblicza czas druku, koszty materiałów i generuje natychmiastowe wyceny.

## Stack technologiczny

- **Backend**: PHP z MySQL
- **Frontend**: JavaScript z własnym UI
- **Przetwarzanie 3D**: Własne algorytmy do analizy plików STL
- **Automatyzacja**: Automatyczne wyliczanie cen na podstawie parametrów modelu

## Funkcje

- Natychmiastowe generowanie wycen
- Optymalizacja kosztów materiałów
- Szacowanie czasu druku
- Wsparcie wielu materiałów

## Wpływ

Skrócenie czasu wyceny z godzin do sekund, znacząco poprawiając doświadczenie klienta i efektywność biznesu.
